# InfraWatch

Projeto de monitoramento de infraestrutura desenvolvido para estudos e prática de backend, infraestrutura, cloud e DevOps.

## Objetivo

Criar um sistema capaz de monitorar máquinas e servidores, exibindo informações como:

- Status online/offline
- Uso de CPU
- Uso de memória RAM
- Uso de disco
- Monitoramento básico de infraestrutura

---

## Tecnologias

### Backend
- Python / FastAPI
- PostgreSQL

### Infraestrutura
- Ubuntu Server
- Docker
- Nginx
- Linux
- Firewall

### Futuramente
- Cloud
- CI/CD
- Monitoramento avançado
- Containers
- DevOps
